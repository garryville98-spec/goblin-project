# $Goblin Investment Platform UI

A React + Vite frontend mockup for **$Goblin**, a crypto investment platform dashboard.

## Included UI sections

- Dashboard with portfolio metrics and market activity
- Top stocks to trade
- Fixed deposit vaults with yield preview
- Stakepool cards
- Launchpad project tracker
- Leverage workspace with simulated risk preview
- User profile and preference settings

## Run locally

```bash
cd goblin-project
npm install
npm run dev
```

Open the URL shown by Vite, usually `http://localhost:5173`.

## Build

```bash
npm run build
```

This project is a frontend UI mockup only. It does not connect to wallets, execute trades, or process deposits.
